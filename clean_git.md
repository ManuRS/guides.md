
## Comandos

```
git checkout --orphan latest_branch
```

```
git add .
```

```
git commit -m "Web"
```

```
git branch -D main
```

```
git branch -m main
```

```
git push -f origin main
```

## Notas 

1. La rama podría llamarse master o main, revisar antes

2. Se necesita el token/contraseña de GitHub o certificado para el commint
